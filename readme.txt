Stop user to submit a web address on the very first comments on any post and send those comments to spam folder.

== Description ==

Using this plugins, blog admin/site owner can stop user, to submit a web address on the very first comments on any post, and  send those comments to spam folder automatically.

== Installation ==

1. Download the plugin and extract the files
2. Upload "anti-spam-comment" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's it, nothing u have to do more. This plugins will automaticaly do its job :)
